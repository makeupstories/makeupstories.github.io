1. 什么是列表 list

列表 list 是 Python 内置的一种数据类型。在[]用逗号分隔开多个任意类型的值，通常用来描述同一种类型的值，放入列表。

list 的特点：

  • 有序
  • 存多个值
  • 可以按索引村取值
  • 可以随时添加和删除其中的元素
  • 可变类型（可变：值可变，id不变）
  • 不可hash
